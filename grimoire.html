⚙️ Git Commands

✶ ✦ ✧ ✵ ✶

  • git status — Check what’s changed.
  • git add . — Stage all changes.
  • git add app.py run.py — Stage specific files.
  • git commit -m "Your message" — Save your staged changes with a message.
  • git push — Send changes to GitHub.
  • git pull origin main --rebase — Pull latest changes before pushing.
  • git push --force — Overwrite remote (⚠️ use with caution).

🧪 Virtual Machine

✶ ✦ ✧ ✵ ✶

🧙 Advanced Terminal Spells

✶ ✦ ✧ ✵ ✶

  • alias gs='git status' — Create a quick alias for a frequent command.
  • history | grep keyword — Search your terminal history for past commands.
  • !! — Repeat the last command.
  • !123 — Run command number 123 from your history.
  • df -h — View disk space usage in human-readable format.
  • top — View real-time system processes.
  • htop — Fancy version of top (requires install).
  • du -sh * — Show folder sizes in current directory.
  • ps aux | grep python — Find running Python processes.
  • curl wttr.in — Get the weather from your terminal.
  • uptime — See how long the system has been running.
  • whoami — Display current user.
  • chmod +x script.sh — Make a script executable.
  • chown user:group file — Change file ownership.
  • scp file user@host:/path — Securely copy files between systems.
  • rsync -avh --progress src/ dest/ — Sync directories with progress display.
  • grep -r "pattern" . — Recursively search for a pattern.
  • find . -name "*.py" — Find all Python files from current directory down.
  • nc -zv host port — Check if a port is open on a host.
  • tar -czvf archive.tar.gz folder — Compress a folder into a tar.gz file.

📦 Package Management Magic

✶ ✦ ✧ ✵ ✶

  • pip install package — Install a Python package.
  • pip list — Show installed Python packages.
  • pip freeze > requirements.txt — Save package list to file.
  • npm install package — Install a Node.js package.
  • npm run dev — Start a development server.
  • npm install -g package — Install globally with npm.
  • sudo apt update — Refresh package list (Debian-based).
  • sudo apt install package — Install packages (Debian-based).
  • sudo dnf install package — Install packages (Red Hat-based).
  • sudo dnf groupinstall "Development Tools" — Install dev tools on Fedora/RHEL.

⚗️ Shell Script Alchemy

✶ ✦ ✧ ✵ ✶

  • #!/bin/bash — Declare the script interpreter.
  • echo "Hello, World!" — Print a message to the terminal.
  • read -p "Enter name: " name — Take user input into a variable.
  • if [ "$name" == "Sage" ]; then echo "Welcome back!"; fi — Basic conditional logic.
  • for file in *.txt; do echo $file; done — Loop over files with a pattern.
  • function greet() { echo "Hello, $1"; } — Define a simple function.
  • chmod +x script.sh — Make your script executable.
  • ./script.sh — Run your shell script.
  • crontab -e — Edit scheduled tasks for automation.
  • trap "echo Exiting..." EXIT — Trap cleanup commands on script exit.
